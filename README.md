QLCoverFlowView
===============

QLCoverFlowView is a scrollable view to display number of page views with cover flow animation. 
It uses enqueue/dequeue mechanism, datasource, delegate which are inspired by UITableView. 

# Requirements
* iOS 5 or later
* QuartCore 
* CoreGraphics
